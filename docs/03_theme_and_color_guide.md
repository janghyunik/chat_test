# 색상 가이드

이번 수정에서는 아래 4가지 색상만 사용하도록 전역 스타일을 정리했습니다.

- `#6A89A7`
- `#BDDDFC`
- `#88BDF2`
- `#384959`

## 어디를 수정하면 되는가

가장 먼저 볼 파일은 아래입니다.

```text
my-app/src/app/globals.css
```

이 파일의 맨 위 `:root` 블록에서 전역 색상 변수를 관리합니다.

```css
:root {
  --color-1: #6A89A7;
  --color-2: #BDDDFC;
  --color-3: #88BDF2;
  --color-4: #384959;
}
```

## 현재 색상 매핑

- `--bg`: 전체 배경
- `--panel`: 카드 / 테이블 / 로그인 박스 배경
- `--panel-soft`: 보조 버튼 / hover 배경
- `--accent`: 주요 버튼 / active 항목
- `--line`: 테두리
- `--text`: 어두운 배경 위 기본 텍스트
- `--text-strong`: 밝은 카드 위 텍스트
- `--muted`: 보조 문구

## 다른 조합을 쓰고 싶을 때

### 방법 1. 색상 원본만 바꾸기
가장 쉬운 방법입니다.

```css
--color-1: 새색상;
--color-2: 새색상;
--color-3: 새색상;
--color-4: 새색상;
```

이 방식은 전체 UI가 한 번에 바뀝니다.

### 방법 2. 역할 매핑만 바꾸기
원본 4개는 유지하고, 어떤 역할에 어떤 색을 쓰는지만 바꿀 수 있습니다.

예:

```css
--bg: var(--color-2);
--panel: var(--color-4);
--accent: var(--color-1);
```

이 방식은 같은 4개 색으로 분위기만 바꾸고 싶을 때 좋습니다.

## 실무 팁

- 전체 배경은 가장 진한 색
- 카드 배경은 가장 밝은 색 또는 중간 색
- 강조 버튼은 채도가 높은 색
- 본문 텍스트는 배경과 대비가 큰 색

으로 잡으면 무난합니다.
